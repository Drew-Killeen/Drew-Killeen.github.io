Most everyone has heard the word encryption and might even have an idea of what it means - it's simply a way to obscure information. Encryption is used all over the place, almost exclusively for security purposes. This page represents the branch of cryptography known as ciphers. This branch is integral to all encryption, and really everything else involving cryptography is based upon it.

A cipher is a specific way to turn a specific message into a set of random letters, and a method to reverse that process in order to decrypt the message. Without knowing the cipher which was used to encrypt a message, it's nearly impossible to decrypt a secure message. However, even if you do know the cipher, it may still be very difficult to break the code. Most ciphers use something known as a key, which is used to encrypt the letters. The key can be anything from a number to an entire sentence itself. The more possibilities that exist for different keys, the more secure a cipher is considered due to it being much harder to guess the correct key.

By using this app, you're able to encrypt messages using a couple different classical ciphers, known as the Caesar Cipher, the Vigenere Cipher, and the Autokey Cipher. Plus, you also get to play with Pig Latin just for the fun of it. If you'd like to learn more about any of the ciphers, each one has a link to a Wikipedia page which covers the algorithm extensively. All default values may be used to create a simple encryption, or more advanced and secure values may be chosen.

This application is intended only for entertainment purposes. Although the plaintext is really encrypted, it only uses classical encrpytion methods, which can be relatively easily decrypted by a computer.
